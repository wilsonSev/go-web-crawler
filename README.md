# Crawler Service

HTTP-сервис на Go для параллельной проверки набора URL-адресов.  
Сервис принимает список ссылок, обрабатывает их с помощью worker pool и возвращает HTTP status code либо ошибку для каждого URL.

## Возможности

- batch-обработка списка URL
- параллельная проверка ссылок через worker pool
- ограничение времени выполнения через `context.WithTimeout`
- кэширование результатов на короткое время
- дедупликация одновременных запросов к одному и тому же URL через `singleflight`
- нормализация URL перед обработкой
- корректное завершение HTTP-сервера через graceful shutdown
